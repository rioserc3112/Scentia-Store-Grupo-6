# Scentia Store – Perfume E-Commerce System

Sistema web transaccional desarrollado para la gestión y venta de **perfumes y decants**.  
El sistema permite a los usuarios explorar productos, agregarlos al carrito y realizar compras, mientras que los administradores pueden gestionar inventario, productos y pedidos.

Este proyecto fue desarrollado como parte del curso **Desarrollo de Aplicaciones Web y Patrones**.

---

# Descripción del Proyecto

Scentia Store es una aplicación web orientada a la venta de perfumes y decants mediante un catálogo digital.  
El sistema permite a los usuarios navegar por productos disponibles, gestionar un carrito de compras y completar pedidos.  
Además, incluye un panel administrativo que permite mantener actualizado el inventario y gestionar los productos del catálogo.

---

# Funcionalidades

## Funcionalidades de Usuario

- Registro de usuario
- Inicio de sesión
- Visualización del catálogo de perfumes
- Consulta de disponibilidad de productos
- Agregar productos al carrito
- Modificar cantidades en el carrito
- Eliminar productos del carrito
- Visualizar subtotal y total de compra
- Confirmar pedidos
- Confirmación visual de compra realizada

---

## Funcionalidades de Administrador

- Acceso al panel administrativo
- Visualización de la lista de productos
- Agregar nuevos productos
- Editar productos existentes
- Desactivar productos
- Modificar stock de inventario
- Visualizar pedidos realizados
- Cambiar estado de pedidos

# Arquitectura

El proyecto sigue el patrón **MVC (Model – View – Controller)**:

- **Model:** Representa las entidades del sistema (usuarios, productos, pedidos).
- **View:** Interfaces de usuario construidas con Thymeleaf.
- **Controller:** Maneja las peticiones HTTP y la lógica de navegación.
- **Service:** Contiene la lógica de negocio del sistema.
- **Repository:** Maneja la comunicación con la base de datos.
